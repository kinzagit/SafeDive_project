# SafeDive_project
Hosten via Streamlit
Installeren van streamlit: pip install streamlit 
controle juiste installatie van streamlit: pip show streamlit

Installeren van spacy: pip install spacy
Installeren van nl package bij spacy: python -m spacy download nl_core_news_sm

Installeren van dateparser: pip install dateparser

Installeren van rapidfuzz ( bib die woorden vergelijkt op basis van gelijkenis in plaats van exacte match)
    Dit om te checken als iemand 1 letter foutief heeft geschreven of ontbreekt dat de woord filter nog correct kan zijn.

Runnen in Console eerste keer met python -m streamlit run SafeDrive_project.py
